API REST — FujiFood

Endpoints disponibles para operaciones CRUD sobre el catálogo de menús. Cada menú contiene: nombre, descripción, precio, stock e imagen (URL).

Método Endpoint Descripción
GET /api/menus Listar todos los menús
GET /api/menus/{id} Obtener un menú por ID
POST /api/menus Crear un nuevo menú
PUT /api/menus/{id} Actualizar un menú
DELETE /api/menus/{id} Eliminar un menú

Modelo del menú

{
  "id":          1,
  "nombre":      "Sushi Set Premium",
  "descripcion": "12 piezas de sushi variado",
  "precio":      28.50,
  "stock":       15,
  "imagen":      "https://ejemplo.com/sushi.jpg",
  "categoria":   "fideos-ramen"
}

Ejemplos de uso

GET Obtener todos los menús 
curl -X GET http://localhost:8000/api/menus/
fetch('http://localhost:8000/api/menus/')
  .then(res => res.json())
  .then(data => console.log(data));
GET Obtener menú por ID 
curl -X GET http://localhost:8000/api/menus/1
fetch('http://localhost:8000/api/menus/1')
  .then(res => res.json())
  .then(data => console.log(data));
POST Crear un menú 
curl -X POST http://localhost:8000/api/menus/ \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Sushi Set Premium",
    "description": "12 piezas de sushi variado",
    "price": 28.50,
    "stock": 15,
    "img": "https://ejemplo.com/sushi.jpg",
    "category": "fideos-ramen"
  }'
const nuevoMenu = {
  name: "Sushi Set Premium",
  description: "12 piezas de sushi variado",
  price: 28.50,
  stock: 15,
  img: "https://ejemplo.com/sushi.jpg",
  category: "fideos-ramen"
};

fetch('http://localhost:8000/api/menus/', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify(nuevoMenu)
})
  .then(res => res.json())
  .then(data => console.log(data));
PUT Actualizar un menú 
curl -X PUT http://localhost:8000/api/menus/1 \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Sushi Set Deluxe",
    "description": "18 piezas de sushi variado",
    "price": 38.00,
    "stock": 10,
    "img": "https://ejemplo.com/sushi-deluxe.jpg",
    "category": "fideos-ramen"
  }'
const datosActualizados = {
  name: "Sushi Set Deluxe",
  description: "18 piezas de sushi variado",
  price: 38.00,
  stock: 10,
  img: "https://ejemplo.com/sushi-deluxe.jpg",
  category: "fideos-ramen"
};

fetch('http://localhost:8000/api/menus/1', {
  method: 'PUT',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify(datosActualizados)
})
  .then(res => res.json())
  .then(data => console.log(data));
DELETE Eliminar un menú 
curl -X DELETE http://localhost:8000/api/menus/1
fetch('http://localhost:8000/api/menus/1', {
  method: 'DELETE'
})
  .then(res => {
    if (res.status === 204) console.log('Eliminado correctamente');
  });

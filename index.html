API REST — FujiFood

Endpoints disponibles para operaciones CRUD sobre el catálogo de menús. Cada menú contiene: nombre, descripción, precio, stock e imágenes.

Método Endpoint Descripción Cuerpo (JSON)
GET /api/menus Obtener todos los menús —
GET /api/menus/{id} Obtener un menú por ID —
POST /api/menus Crear un nuevo menú { nombre, descripcion, precio, stock, imagen }
PUT /api/menus/{id} Actualizar un menú { nombre, descripcion, precio, stock, imagen }
DELETE /api/menus/{id} Eliminar un menú —
